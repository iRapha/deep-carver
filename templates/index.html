What it is

Deep Carver brings machine learning to seam carving

But what is Seam Carving?

Its the algorithm that PhotoShoop uses for content aware scaling.

The algorithm recognizes seams in the image that seem similar to their neighbors and then removes them to give a more seamless (pun intended) result when scaling.

So what did you do?

Seam carving isn't perfect, sometimes things like faces can be targeted as unimportant seams which can lead to unintended results.

So we are tyring to fix it with machine learning

So we are using machine learning to fix this without human intervention
